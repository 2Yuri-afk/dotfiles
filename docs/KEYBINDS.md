# ⌨️ Keybindings Reference

## 🪟 Window Management

| Keybind | Action |
|---------|--------|
| `Super + Q` | Close active window |
| `Super + V` | Toggle floating mode |
| `Super + F` | Toggle fullscreen |
| `Super + Shift + Q` | Force kill window |
| `Super + M` | Exit Hyprland |

## 🚀 Applications

| Keybind | Action |
|---------|--------|
| `Super + Return` | Terminal (Kitty) |
| `Super + D` | App launcher (Rofi) |
| `Super + Shift + D` | Scripts menu |
| `Super + E` | File manager (Thunar) |
| `Super + B` | Browser |

## 🎮 Workspaces

| Keybind | Action |
|---------|--------|
| `Super + 1-9` | Switch to workspace 1-9 |
| `Super + Shift + 1-9` | Move window to workspace 1-9 |
| `Super + Mouse_Wheel` | Cycle through workspaces |
| `Super + Tab` | Cycle through windows |

## 🎵 Media Controls

| Keybind | Action |
|---------|--------|
| `XF86AudioPlay` | Play/Pause |
| `XF86AudioNext` | Next track |
| `XF86AudioPrev` | Previous track |
| `XF86AudioRaiseVolume` | Volume up |
| `XF86AudioLowerVolume` | Volume down |
| `XF86AudioMute` | Toggle mute |

## 🔆 System Controls

| Keybind | Action |
|---------|--------|
| `XF86MonBrightnessUp` | Brightness up |
| `XF86MonBrightnessDown` | Brightness down |
| `Super + L` | Lock screen |
| `Super + P` | Power menu |

## 📸 Screenshots

| Keybind | Action |
|---------|--------|
| `Super + Shift + S` | Screenshot menu |
| `Print` | Full screen screenshot |
| `Super + Print` | Window screenshot |
| `Shift + Print` | Region screenshot |

## 🎯 Rofi Menus

| Keybind | Action |
|---------|--------|
| `Super + D` | Application launcher |
| `Super + Shift + D` | Scripts menu |
| `Super + P` | Power menu |
| `Super + Shift + S` | Screenshot menu |

## 🐭 Mouse Actions

| Action | Result |
|--------|--------|
| `Super + Left_Click` | Move window |
| `Super + Right_Click` | Resize window |
| `Super + Mouse_Wheel` | Change workspace |

## 🔧 Special Functions

| Keybind | Action |
|---------|--------|
| `Super + R` | Reload Hyprland config |
| `Super + Shift + R` | Restart Waybar |
| `Super + N` | Toggle notifications |
| `Super + Shift + N` | Clear all notifications |

---

💡 **Tip**: All keybinds can be customized in `~/.config/hypr/hyprland.conf`
